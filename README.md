# Finance Controller (React Native)

This repository now keeps only the React Native codebase (Expo + TypeScript).

## Workspace Structure

- `apps/mobile`: mobile application
- `packages/shared`: shared types/validation/utils used by mobile
- `supabase`: SQL and setup resources

## Requirements

- Node `>=20.19.4`
- npm `>=10`
- Android Studio (for Android emulator/device builds)

## Install

```bash
npm install
```

## Run Mobile App

```bash
npm run mobile:dev
```

Other useful commands:

```bash
npm run mobile:android
npm run mobile:ios
npm run mobile:web
```
