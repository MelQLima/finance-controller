# Finance Controller Mobile

Expo Router mobile app scaffold for the existing Finance Controller project.

## Setup

1. Copy `.env.example` to `.env`.
2. Fill `EXPO_PUBLIC_SUPABASE_URL` and `EXPO_PUBLIC_SUPABASE_ANON_KEY`.
3. Install from repo root with `npm install`.
4. Run from repo root with `npm run mobile:dev`.
